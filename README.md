# E-commerce Sales & Customer Insights Dashboard

## Overview
This project analyzes e-commerce sales data to identify sales trends, customer behavior, and business insights using Python and Power BI.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Power BI

## Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Sales Trend Analysis
- Customer Insights
- Regional Performance Analysis
- Interactive Power BI Dashboard

## Files Included
- sales_analysis.ipynb → Data cleaning and analysis
- ecommerce_dashboard.pbix → Power BI dashboard
- sales_data.csv → Dataset used for analysis
